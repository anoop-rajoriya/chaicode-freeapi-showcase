# Quotes Generator [Live](https://chaicode-quotes-generator.netlify.app/)

A lightweight quote generator built with Vite and React. This project fetches quotes from a free API and displays them with a simple loading and error experience.

## Tech

- Vite
- React
- JavaScript
- CSS

## Local setup

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start the development server:
   ```bash
   pnpm dev
   ```
3. Open the app in your browser at the local URL shown in the terminal.

Enjoy building and customizing the quote experience!
