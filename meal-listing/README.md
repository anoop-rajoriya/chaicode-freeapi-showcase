# Meal Listing [Live](https://chaicode-meal-listing.netlify.app/)

## Overview

A simple meal listing application built with React that displays a list of meals with pagination.

## Techstack

- React
- Vite
- Tailwind CSS
- ESLint
- pnpm

## Local Setup

1. Install dependencies: `pnpm install`
2. Start the development server: `pnpm dev`
