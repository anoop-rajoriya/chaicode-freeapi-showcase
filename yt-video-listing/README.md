# YouTube Video Listing [Live](https://chaicode-yt-video-list.netlify.app/)

## Overview

A simple React + Vite app that fetches and displays YouTube-style video listings using a free API. The project shows how to build a responsive video catalog with search and loading states.

## Tech Stack

- Vite
- React
- JavaScript
- CSS

## Local Setup

1. Clone the repository.
2. Open the `yt-video-listing` folder.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open the URL shown in the terminal to view the app.
